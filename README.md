# Clips OS

**Growth Engine for modern Founders and Marketers.**

On-demand conversion optimization audits, campaign analysis, competitor intelligence, and unit economics for DTC e-commerce founders and growth teams.

## Install

```bash
git clone https://github.com/gbrando93/clips-os.git
```

## CRO Audit

The flagship skill. Drop any ecommerce URL and get:

- Page-by-page analysis (Homepage, Collection, PDP, Cart, Checkout, Search)
- Desktop + mobile screenshots with annotated observations
- 1-5 severity scoring per criterion (30+ evaluation points)
- Prioritized action plan (P0-P3) with quick wins, medium-term, and strategic items
- Professional HTML report with color-coded findings

Built on the LIFT model + Baymard research heuristics, tuned for DTC brands (subscription flows, post-purchase upsells, loyalty programs, brand storytelling).
